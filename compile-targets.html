🔧 MindAR Target Compiler

Step 1: Online Compiler (Recommended)

The easiest way to compile targets is using the official MindAR online compiler:

  1. Go to MindAR Target Compiler
  2. Upload all 6 target images from assets/targets/:
    • challenge0.png
    • challenge1.png
    • challenge2.png
    • challenge3.png
    • challenge4.png
    • challenge5.png
  3. Set "Dimensions" to 1
  4. Click "Compile"
  5. Download targets.mind
  6. Place it in assets/targets/targets.mind

Step 2: Alternative - Use Demo Mode

If you cannot compile targets, the app will automatically fall back to Quiz Mode which works without AR.
